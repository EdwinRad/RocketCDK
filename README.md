# Usage
<!-- usage -->
```sh-session
$ npm install -g rocketcdk
$ rocketcdk COMMAND
running command...
$ rocketcdk (-v|--version|version)
rocketcdk/0.0.0 darwin-x64 node-v13.10.1
$ rocketcdk --help [COMMAND]
USAGE test
  $ rocketcdk COMMAND
...
```

## `rocketcdk help [COMMAND]`

display help for rocketcdk

```
USAGE test
  $ rocketcdk help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
